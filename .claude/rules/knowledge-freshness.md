# Knowledge Freshness Principle

After any action that changes the world (code edits, hypothesis resolution, user corrections),
check if knowledge tree nodes need updating. Use Edit (not Write). Update `last_updated` +
`last_update_trigger` front matter. Record debt in working memory if immediate reconciliation
is not possible.

**Detail:** Read `core/config/conventions/infrastructure.md` for full reconciliation protocol, trigger types, and debt schema.
